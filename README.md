🚀 Overview
BuckleUp is a modern and minimal To-Do application built using React.
It helps users organize daily tasks, track productivity, and stay focused.
“Buckle up and get things done.” 💪
✨ Features
➕ Add new tasks
❌ Delete tasks
✅ Mark tasks as completed
🔄 Real-time UI updates
📱 Responsive design
⚡ Fast performance with Parcel bundler
🛠️ Tech Stack
React
JavaScript (ES6+)
HTML5
CSS3
Parcel Bundler
